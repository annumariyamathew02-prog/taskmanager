# Task Manager Expert System

A web-based Task Manager Expert System that helps users organize tasks across different life areas (Work, Personal, Fitness, Shopping, and more) with categories, statuses, and due dates.

## Features

- Add tasks with:
    - Title
    - Description / notes
    - Category (Work, Personal, Fitness, Shopping, Other, etc.)
    - Due date
- View tasks by status:
    - All
    - Completed
    - Pending
- Mark tasks as completed or pending.
- Edit existing tasks.
- Delete tasks you no longer need.
- Clear and user-friendly interface with icons for quick recognition.


## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)


## How To Run

1. Clone or download this repository.
2. Open `index.html` in a modern web browser.
3. Use the category buttons and status filters (All / Completed / Pending) to manage your tasks.

## Project Structure

- `index.html` – Main interface of the Task Manager Expert System.
- `style.css` – Styling for layout, colors, and responsive design.
- `script.js` – Handles task creation, editing, completion, deletion, filtering, and category logic.


## Possible Future Enhancements

- Persist tasks using `localStorage` or a backend API so they remain across sessions.
- Add search by task title or description.
- Add notifications or reminder pop-ups before due dates.
